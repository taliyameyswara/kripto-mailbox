## Cara Menggunakan!

- Buka file `db.php` di folder `include`
- Buka `phpmyadmin` di browser
- Buat database
- Impor database `mail.sql` di folder `include`
- Setting database, samakan server, useer, pasword, dan database kamu
- Siap digunakan!
